title: o-
description: A small JavaScript runner with multiple engine backends.
---

`o-` is a small JavaScript runner that can execute the same script on multiple
engine backends:

- `spidermonkey`
- `javascriptcore`
- `v8`

The project also exposes a shared `jstd` bootstrap layer so basic built-ins
such as `console.log(...)` behave consistently across engines.
